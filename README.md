## Project

<br>

### 프로잭트 소개

- 현재 접속하고 있는 위치의 날씨 정보를 축력
- 검색을 통해 입력한 도시/나라의 날씨 정보를 출력
- 날씨 데이터를 바탕으로 해당 위치의 날씨에따라 배경화면이 변경
- 즐겨찾기 기능을 추가하여 다시 페이지에 접속하더라도 클릭만으로 원하는 지역 날씨 정보 확인 가능
